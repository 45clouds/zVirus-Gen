# zVirus Gen
Those who don't know how to use VCRT, VBug or VBug Map

## Installation:
__Ubuntu__
> apt-get install git -y

__Termux__
> pkg install git -y

__Other__
> apt install git -y

> git clone https://github.com/ZechBron/zVirus-Gen

> cd zVirus-Gen

> chmod +x setup.sh

> ./setup.sh


## To Run:
> ./zVirus

__Or__

> bash zVirus

## To Update
> ./update.sh

__Or__

> bash update.sh

### Note:
It is recommended to update this tool everyday.
